# NIDS Web UI - واجهة ويب للتجربة والتعديل

تمت إضافة واجهة ويب بجانب تطبيق سطح المكتب حتى تستطيع رؤية الواجهة وتعديلها بسرعة.

## التشغيل
من جذر المشروع:

```bash
python web_app/server.py --host 0.0.0.0 --port 7860
```

ثم افتح المتصفح على:

`http://localhost:7860`

## ماذا تقدم؟
- اختيار المودل الجاهز.
- إدخال الخصائص (CSV).
- زر تعبئة عشوائية للتجربة.
- تشغيل كشف الهجوم وعرض نتيجة JSON.

## ملاحظات
- إذا كان PyTorch موجودًا: يعمل التنبؤ الحقيقي.
- إذا لم يكن PyTorch موجودًا: يعمل Mock Mode تلقائيًا لعرض الواجهة والتعديل عليها.
